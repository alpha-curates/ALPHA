Features

Everything your home server needs

A complete operating system for your Raspberry Pi — from AI chat to disk management.

🤖

AI Studio

Chat with local or cloud AI models, generate code, analyze files, create images — all from one interface.

💾

Storage Manager

Monitor drives, SMART data, ZFS pools, RAID status. Safe auto-detection excludes the OS drive.

📱

Device Hub

Auto-discover devices via ARP + nmap, organize by room, ping check, 29 device type icons.

📊

Live Dashboard

Real-time CPU, RAM, disk charts, top processes, alerts, network monitoring, animated counters.

🔧

System Tools

100+ tools across 20 categories: services, Docker, cron, firewall, sensors, USB, kernel modules, packages, and more.

👥

User Management

Granular per-user permissions across 14 pages × 40+ actions. Feature toggles, storage limits, restrictiveness meter.

🎨

Full Customization

10,000+ options: 24 accent colors, 54 built-in wallpapers, custom image upload, typography, layout, and custom CSS.

🔒

Security & Backup

Encryption, backup jobs, audit logs, share links with passwords, granular permissions, and more.
