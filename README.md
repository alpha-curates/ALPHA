## Setup ##

The code below is for a quick, one shot installation
```
curl -sL https://raw.githubusercontent.com/vector-just-technology/Home-server/master/install.sh | bash
```
Once a user logs-in for the first time, they are considered administrator. That can be changed anytime in settings.

### Ollama integration ( optional ) ###
```
curl -fsSL https://ollama.com | sh
ollama run llama3.2:1b
```
Uses 6GB storage, and requires a Raspberry Pi 5 6GB+ ( The server also includes open / wifi models )
